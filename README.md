# 🌱 GreenStream Solutions — E-Waste Management & Sustainability Website

GreenStream Solutions is a platform dedicated to promoting **e-waste collection, recycling, and environmental sustainability**.  
This website is built with **TailwindCSS** for speed and responsiveness, and can easily be connected to a Django CMS backend if needed.

---

## 🚀 Features

- 🏠 **Home Page** – Clean, modern landing page  
- ℹ️ **About Section** – Who we are & what we do  
- ♻️ **Programs** – E-waste collection, recycling, education & advocacy  
- 📢 **Awareness Campaigns** – Driving sustainability conversations  
- 💬 **Contact Section** – Easy contact form & WhatsApp link  
- 🌗 **Dark/Light Mode** – Smooth toggle for better accessibility  
- 💳 **Donate Button** – M-Pesa number (no API required, copy-and-paste friendly)

---

## 🛠️ Tech Stack

- **Frontend:** HTML5 + TailwindCSS
- **Backend (Optional):** Django (CMS Ready)
- **Version Control:** Git + GitHub
